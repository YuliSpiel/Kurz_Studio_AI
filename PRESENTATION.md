---

## Slide 1: 프로젝트 개요

### Kurz AI Studio 🎬
**프롬프트 입력만으로 완성된 숏폼 영상을 자동 생성하는 AI 서비스**

#### 핵심 가치
- **완전 자동화**: 기획 → 이미지 생성 → 음성 합성 → 배경음악 → 영상 합성
- **다중 Agent 협업**: 6개 전문 AI Agent가 역할 분담
- **3가지 모드**: Story / General / Ad Mode
- **마케팅 특화**: 광고 콘텐츠 빠른 제작

## Slide 3: 핵심 기능

### 1. General Mode (일반 모드)
- 프롬프트만으로 자동 시나리오 생성
- 정방형 통합 이미지 (1:1, 1080x1080)
- 빠른 생성 (평균 2-3분)

### 2. Story Mode (스토리 모드)
- 캐릭터 중심 스토리텔링
- 캐릭터 초상화 + 배경 분리 렌더링
- 표정/포즈/위치 제어

### 3. Ad Mode (광고 모드) ⭐
- **마케팅 특화 모드**
- 제품/서비스 홍보 영상
- **제품 링크 추가**만 하면 자동 생성
- 광고 CTA 자동 삽입

### 4. 완전 자동화
- 이미지: Gemini 2.5 Flash Image
- 음성: ElevenLabs TTS (한국어)
- 배경음악: ElevenLabs Music
- 영상 합성: MoviePy (9:16, 30fps)

## Slide 4: 기술 스택

### Backend Architecture
```
┌─────────────────────────────────────────┐
│          FastAPI (Python 3.11)          │
│  - RESTful API                          │
│  - WebSocket (실시간 진행률)            │
└─────────────────────────────────────────┘
                  ↓
┌─────────────────────────────────────────┐
│      Celery + Redis (Task Queue)        │
│  - 비동기 작업 처리                      │
│  - 병렬 처리 (Chord Pattern)            │
└─────────────────────────────────────────┘
                  ↓
┌─────────────────────────────────────────┐
│          6 Specialized Agents           │
│  기획자 → 디자이너 → 작곡가 → 성우      │
│         → 감독 → QA                     │
└─────────────────────────────────────────┘
```

### Core Technologies
| 분야 | 기술 |
|------|------|
| **LLM** | Gemini 2.5 Flash |
| **Image Generation** | Gemini 2.5 Flash Image |
| **TTS** | ElevenLabs API |
| **Music** | ElevenLabs Sound Effects |
| **Video** | MoviePy (Python) |
| **State Management** | Redis FSM |
| **Background Removal** | rembg (U2Net) |

### Frontend
- React + TypeScript + Vite
- Tailwind CSS
- Real-time WebSocket updates

## Slide 6: 워크플로우 (FSM State Machine)

### Finite State Machine
```
IDLE (대기)
  ↓ POST /v1/runs
PLOT_GENERATION (기획자 Agent)
  - LLM이 시나리오 생성
  - characters.json, plot.json, layout.json 생성
  ↓
ASSET_GENERATION (병렬 처리)
  ├─ Designer: 이미지 생성 (Gemini 2.5 Flash Image)
  ├─ Composer: 배경음악 생성 (ElevenLabs)
  └─ Voice Actor: 음성 합성 (ElevenLabs TTS)
  ↓ (Celery Chord Callback)
VIDEO_COMPOSITION (감독 Agent)
  - MoviePy로 영상 합성
  - 자막, 배경음악, 음성 동기화
  ↓
QA (품질 검수 Agent)
  - 오디오/비디오 품질 체크
  - 실패 시 재생성 (최대 3회)
  ↓ (통과)
END (완료)
```

### 평균 처리 시간
- **Story Mode**: 4-6분
- **General/Ad Mode**: 2-3분
- **병렬 처리로 30% 시간 단축**

## Slide 7: Agent별 역할 분담

### 1. 기획자 Agent (Planner)
- **입력**: 사용자 프롬프트, 모드, 장면 수
- **역할**: LLM 기반 시나리오 생성
- **출력**: `characters.json`, `plot.json`, `layout.json`

### 2. 디자이너 Agent (Designer)
- **입력**: `layout.json`의 이미지 프롬프트
- **역할**: Gemini 2.5 Flash Image로 이미지 생성
- **출력**: `images/*.png` (캐릭터 + 배경 or 통합 이미지)

### 3. 작곡가 Agent (Composer)
- **입력**: BGM 프롬프트 (분위기, 장르)
- **역할**: ElevenLabs Music로 배경음악 생성
- **출력**: `audio/global_bgm.mp3`

### 4. 성우 Agent (Voice Actor)
- **입력**: 대사 텍스트, 캐릭터 음성 ID
- **역할**: ElevenLabs TTS로 한국어 음성 합성
- **출력**: `audio/{scene}_{line}.mp3`

### 5. 감독 Agent (Director)
- **입력**: 모든 assets (images, audio)
- **역할**: MoviePy로 영상 합성 (자막, 싱크)
- **핵심 기능**:
  - BGM 자동 루프: 영상 길이보다 짧은 BGM 자동 반복 재생
  - 오디오 믹싱: 배경음악 + 음성 + SFX 동시 처리
  - 타임라인 동기화: 이미지, 자막, 음성 정확한 싱크
- **출력**: `final_video.mp4` (1080x1920, 30fps)

### 6. QA Agent
- **입력**: 완성된 영상
- **역할**: 품질 검수 (해상도, 오디오, 싱크)
- **출력**: 검수 리포트 (통과/재생성)
